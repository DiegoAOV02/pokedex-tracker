## Getting Started

### Prerequisites

This project has a `Makefile` file that uses `pnpm` package manager

```bash
MacOS (homebrew)
brew install pnpm

Windows
iwr https://get.pnpm.io/install.ps1 -useb | iex
```

### Installation

1. Clone repo

```bash
git clone https://github.com/DiegoAOV02/pokedex-tracker.git
```

2. Install pnpm packages with Makefile

```bash
Make install
```
